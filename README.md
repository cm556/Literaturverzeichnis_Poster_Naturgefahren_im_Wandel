# Literaturverzeichnis_Poster_Naturgefahren_im_Wandel
Quellenverzeichnis für das Poster "Bodenfeuchte als Proxy für Flächenbrände auf Sizilien?", welches im Rahmen des Seminars "Naturgefahren im Wandel" am Geographischen Institut der Universität Freiburg erstellt wurde. 

## Daten
- GWIS (JRC/European Commission): Global Wildfire Information System – Burned Area / Fire Perimeters. URL: https://gwis.jrc.ec.europa.eu/apps/country.profile/downloads. (Zugriff: 12.01.2026).
- ERA5 (Copernicus/ECMWF): Monthly data (2m temperature, total precipitation, volumetric soil water layers 1–4), 2002–2023. URL:https://cds.climate.copernicus.eu/datasets/reanalysis-era5-land?tab=download. (Zugriff: 12.01.2026).
